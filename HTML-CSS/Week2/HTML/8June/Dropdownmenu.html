##### overflow The overflow property specifies whether to clip the content or to add scrollbars when the content of an element is too big to fit in the specified area. The overflow property has the following values: 1. visible - Default. The overflow is not clipped. The content renders outside the element's box 2. hidden - The overflow is clipped, and the rest of the content will be invisible 3. scroll - The overflow is clipped, and a scrollbar is added to see the rest of the content 4. auto - Similar to scroll, but it adds scrollbars only when necessary ##### CSS layout- horizontal and vertical align **Center align element**
When we set margin: auto then this element will be aligned at center.
